## Installation

You can install the latest released version of **pyexplorer** using `pip`.

`pip install pyexplorer`

Alternativaly you can install it from the source code.

```bash
git clone git@github.com:dexpota/pyexplorer.git
cd pyexplorer
pip install -e .
```
